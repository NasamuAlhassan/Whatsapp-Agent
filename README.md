# WhatsApp Agent Dashboard

A personal AI-powered WhatsApp monitoring agent with a real-time glass dashboard.

## Features
- Real-time message monitoring via WebSocket
- AI-powered urgency detection & digest summaries (Groq)
- Telegram notifications for urgent messages
- Task & news extraction from group chats
- Natural language "Ask AI" queries
- Light / dark mode glass UI
